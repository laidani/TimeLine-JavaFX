# TimeLine-JavaFX
Basic idea about a TimeLine using JavaFX

In this example we use Java 8, jfoenix-1.9.0, Netbeans, Scene Builder, lombok.

### Prerequisites

This project use 
  - Java 8
  - jfoenix-1.9.0
  - Any IDE like Netbeans Eclipse IntelliJ IDEA 
  - Scene Builder.
